# 🎮 Rock Paper Scissors Game (Python)
# 📌 Description

This is a simple Rock Paper Scissors game built using Python. The user plays against the computer, and the winner is decided based on classic game rules.

# ⚙️ Features
User vs Computer gameplay
Random computer choice
Score display
Simple command-line interface
# 📜 Rules
Rock beats Scissors
Scissors beats Paper
Paper beats Rock
# Technologies used
Python
